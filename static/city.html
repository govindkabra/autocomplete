This demo provides autocomplete API over all cities in the world. It is based on this github project.

It supports search by prefix of every term: for "pa", it will match Paris, Patna; as well as, Sao Paulo and El Paso. But it does not do substring searches, e.g., "pa" would not match Bhopal.
